**Inversion:** USD 6,000

### Entregables

| # | Entregable |
|---|---|
| 1 | Base de datos y migraciones |
| 2 | Backend/API funcional con todas las validaciones |
| 3 | Tienda online mobile-first (catalogo, carrito, checkout) |
| 4 | Panel admin (pedidos, productos, categorias, configuracion) |
| 5 | Testing funcional end-to-end |
| 6 | Despliegue a produccion con dominio, SSL y backups |

### Que incluye

- Catalogo por categorias con imagenes
- Carrito de compras con subtotales
- Checkout: nombre, telefono, retiro o envio
- Confirmacion de pedido con ID y estado
- Bandeja de pedidos por estado (nuevo, en preparacion, listo, enviado, cerrado)
- CRUD de productos y categorias
- Configuracion de horarios, pausa manual, envio (fijo o por zona)
- Autenticacion segura para admin
- Link a WhatsApp para consultas

## Condiciones

- Cada fase tiene alcance cerrado. No se compromete una fase futura hasta cerrar la anterior.
- Cambios de alcance se presupuestan como adicionales.
- Discovery es descontable del total si se contrata el proyecto completo.
- Garantia de 30 dias de correccion de errores por fase (no incluye nuevas funcionalidades).
- Soporte post-garantia requiere plan mensual.
